Vison Statement: Beers, Wines and Ciders

The goal of this application is to allow a user to buy and receive alchol. It will basic and easy to use and allow a user to 
travel through the application while adding the drinks they want to by to their basket. 
It will allow an adminstrator to manage the system by making modifications to the system like adding new stock, removing stock
or as simple as a stock enquiry to check the stock level.
